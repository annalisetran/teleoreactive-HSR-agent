## Database

This folder contains setup scripts for the world model database.

| File | Description |
|------|-------------|
| setup.sh | Bash script that runs all of the setup processes |
| setup_drop_tables_x.x.x | Script to drop all of the tables and functions |
| setup_schema_x.x.x | Script to create the database schema, including table, foreign keys, etc. x.x.x refers to the version of the database |
| setup_reference_data_.x.x.x | Inserts the reference data |
| setup_insert_data_.x.x.x | Insert some sample data into the database |
