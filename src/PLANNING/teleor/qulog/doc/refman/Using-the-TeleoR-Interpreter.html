3.4 Using the TeleoR Interpreter

This is invoked using the command Teleor with exactly the same option flags as the QuLog interpreter. Its queries and commands are an extension of QuLog interpreter queries and commands. Its query prompt is | ?~>.

Using the TeleoR interpreter, files can be consulted containing TeleoR procedure definitions. It also supports extra action primitives start_agent and start_task. start_agent will launch a generic robotic agent shell interacting with a robot interface process or simulator. start_task will create tasks within the agent each executing aTeleoR procedure call.

If Pedro is being used for communication then the TeleoR command needs to be followed by the -A switch in order to connect to Pedro and give a name to the agent.
